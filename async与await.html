    async与await

    临风 3/13/2022 es6+异步

    async/await是ES8引入的新语法，用来简化Promise异步操作。在此之前，开发者只能通过链式.then()处理 Promise异步操作

    # async

    async 是“异步”的简写，async 用于申明一个 function 是异步的。

    • async 函数返回的是一个 Promise 对象。async 函数（包含函数语句、函数表达式、Lambda表达式）会返回一个 Promise 对象。如果在函数中 return 一个直接量，async 会把这个直接量通过 Promise.resolve() 封装成 Promise 对象。如果 async 函数没有返回值，它会返回 Promise.resolve(undefined)。
    • async可以单独使用，所以在没有 await 的情况下执行 async 函数，它会立即执行，返回一个 Promise 对象，并且，绝不会阻塞后面的语句。这和普通返回 Promise 对象的函数并无二致。
    • 在async方法中，第一个await之前的代码会同步执行，await之后的代码会异步执行

    # await

    await 可以认为是 async wait 的简写, await 用于等待一个异步方法执行完成。

    • await等的实际是一个返回值。不仅仅用于等待Promise对象，它可以等任意表达式的结果
    点击查看代码 
    function getSomething() {
    return "something";
}

async function testAsync() {
    return Promise.resolve("hello async");
}

async function test() {
    const v1 = await getSomething(); 
    const v2 = await testAsync();
    console.log(v1, v2);
}

test(); // 完全可以正常运行
    • 如果await等待一个Promise对象，则会阻塞后面代码。await会等着Promise对象resolve，然后得到resolve的值，作为await表达式的 运算结果。await 等待的不是一个 Promise Like 对象的时候，相当于 await Promise.resolve(...)。
    • 为使await的调用不会造成阻塞，则需放入async函数中异步执行，将他内部所有的阻塞都被封装在一个Promise对象中异步执行
    • await 一个 Promise 只能接受到成功resolve()的结果，对于 reject() 的结果会报错。可通过 try catch 来处理

    # 注意点

    一个aync函数中不建议放多个await 函数，应为第一个await 会阻塞后面的代码，影响其他代码的执行。如果要使用多个await，建议使用Promise.all

    async function f(){
    const promisea = await fetch("http://.../post/1");
    const promiseb = await fetch("http://.../post/1");// 要等到第一个执行完后才会开始执行第二个任务

    const [a,b] = await Promise.all([promisea, promiseb]) //此写法会提升一倍效率
}

    如果我们需要在循环中执行异步操作，不能直接调用 forEach 或者 map 这一类方法,这类方法不会等待 await 执行完成，而是立即返回。如果我们希望等待循环中的异步操作都一一完成后才继续执行，应当使用传统的for循环，如果想要所有循环中的操作都并发执行。 可以使用 for await

    async function f(){
    [1,2,3].forEach(async (i) =>{
        await someAsyncOperation()
    })
    console.log("done") //先输出 done
}
f()

    async function f(){
    for (let i of [1,2,3]){ //传统的for循环
        await someAsyncOperation()
    }
    console.log("done") //先执行异步函数，异步执行结束后才输出done
}

    async function f(){//并发执行异步操作
    const promises = [ 
        someAsyncOperation(),
        someAsyncOperation(),
        someAsyncOperation()
    ]

    for await (let result of promises){
        // ...
    }
    console.log("done") // 先同步执行所有异步函数，等所有异步函数都执行完后，才输出 done
}
